# Portafolio · Arte & Cultura

Portafolio estático para la materia Arte & Cultura (EIA 4, INBAL 2026).

Publicado en **https://sk-dv.github.io/portafolio-arte/**.

## Contenido

**`index.html`** — página principal del portafolio, organizada en dos unidades:

- *Unidad I* — Cultura, arte, estética y percepción
- *Unidad II* — Interpretación: imagen, relato y lenguajes del arte

**`las_vegas_tango.html`** — presentación independiente sobre *Las Vegas Tango* de Gil Evans (1964).
Cubre la pieza, el álbum, los músicos y su lugar dentro del Cool Jazz, el jazz modal-orquestal y el Third Stream.
Enlazada desde el índice mediante ruta relativa.

## Stack

HTML + CSS puro. Sin build step ni dependencias.
Fuentes cargadas desde Google Fonts CDN (Fraunces + Newsreader para el portafolio; Playfair Display + Inter para la presentación).
